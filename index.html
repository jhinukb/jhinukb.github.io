I Create Predictive Models

Being a data scientist/analyst, I can develop any model from data to present findings. 80% of my work involves getting my hands dirty with cleaning dirty data. Data comes in all sorts of skewed formats (not always csv or json files) which can be difficult to manipulate. My next task is to visualize this data and find trends. I can then use this to develop models (linear regression, feature engineering...etc) to find results/statistically significant correlations in the data.

  Clean Code

I write functions to clean tables in Pandas and can generate SQL queries to extract certain features of the table. I use regular expressions for NLP tasks, remove null values, make sure all values are the same data type and more.

  Beautiful Visualizations

I generate data visualizations whether it's categorical data (bar graphs) or numerical data (scatterplots, line plots, time series). I can achieve this using Python libraries such as Matplotlib and Seaborn or Business Intelligence tools such as Google Big Data Studio or Tableau.

  Models

I apply ML and AI techniques by designing classification models, regression models, neural networks and to analyze the data based on the trends I have found. This part can be challenging but is the most intriguing to me.

  Delivery

Many data scientists are highly analytical and only focus on the analytical aspect. However, delivering these findings to the non-technical clients is extremely important. I have experience presenting data/findings and writing about it in professional manner.
